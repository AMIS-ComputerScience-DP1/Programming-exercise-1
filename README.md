# Programming-exercise-1

## Instructions

### Step 1: Create your file
1.  Open **Thonny** on your computer.
2.  Create a new file and save it as `main.py`.

### Step 2: Write the Code
You cannot just write the joke in one line! You must use two lines minimum.

### Example Output
When you run your code, your screen should look something like this:

```text
Why do Python programmers wear glasses?
Because they can't C#.
```

### Submission
You can submit your work in two ways. Choose the one that is easiest for you!

### Option A: Upload your file (Recommended)
1.  Save your `main.py` file on your computer.
2.  Go to the top of this page, click **"Add file"**, and select **"Upload files"**.
3.  Drag your `main.py` file into the box.
4.  Click the green **"Commit changes"** button at the bottom.

### Option B: Copy and Paste
If you prefer, you can write the code directly on GitHub:
1.  Open your code in Thonny and copy all the text.
2.  Go to the top of this page, click **"Add file"**, and select **"Create new file"**.
3.  Name the file `main.py` in the box at the top.
4.  Paste your code into the large text box.
5.  Click the green **"Commit changes"** button at the bottom.

### How to check if you are done
Refesh this page. If you see `main.py` in the file list above, you have finished!
